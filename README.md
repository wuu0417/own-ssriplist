# 项目简介
自己创建的ShadowsocksR Plus+分流列表，包含**Netflix、OpenAI** IP段。

SSRPlus的Netflix分流功能默认只支持Netflix/Netflix+AWS双选项，该列表可用来自定义平替。

链接：https://raw.githubusercontent.com/wuu0417/own-ssriplist/main/ownlist.txt
